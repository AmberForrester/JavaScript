Removing negative numbers from an array list using the map( ) method

The map ( ) method is primarily used for transforming elements of an array to create a callback function.

It is not the ideal choice for filtering out elements based on a condition like removing negative numbers.

You would use the filter ( ) method instead, but you could combine both methods if you wanted to use map ( ). Together with the filter ( ) method.

As shown in the console log - map ( ) is used to create a new array that's identical to the original one and maintains the original array's length and positions.

By chaining the filter ( ) method to the mapped array. We can filter out only the elements that are greater than or equal to zero, effectively removing the negative numbers.

The filter( ) method on its own works by using this syntax: const positiveNumbers = numbers.filter (num => num >= 0);
